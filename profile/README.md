# Knowage Labs

**Knowage** is the professional open source suite for modern business analytics
over traditional sources and big data systems.

Developed and maintained by [Engineering Ingegneria Informatica](https://www.eng.it),
Knowage is part of the [OW2](https://www.ow2.org) open source community.
